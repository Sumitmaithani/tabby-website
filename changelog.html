Changelog

v1.1 — 21 may 2026

  • Import bookmarks from Chrome, Arc, Brave, Edge, and Comet in one click.
  • Import from file works too: CSV or JSON.
  • New first-launch screen: choose browser import, choose file import, or skip.
  • You can preview before saving. Tabby shows duplicate counts first.
  • Auto-backup now runs before every import. If an import goes sideways, roll back.
  • Fixed ⌘⇧L on fresh installs.
  • Popover now comes to the front when opened, so typing works immediately.
  • Fixed a bug where "add bookmark" sometimes opened with stale text from a previously edited link.
  • Removed the light/dark/system picker. Tabby follows system appearance now.
  • Fixed a small memory leak in the hotkey handler.

v1.0 — 17 may 2026

  • One keystroke to save a link: ⌘⇧L from anywhere.
  • Paste a URL, Tabby pulls title and favicon.
  • Search titles, URLs, tags, and notes as you type.
  • Color tags. Sort by date, name, or how often you click something.
  • Export to CSV, JSON, HTML, Markdown, or PDF.
  • Everything is in one JSON file on your Mac. No account. No cloud. No telemetry.
